# Affective Pedagogical Agent Motion Dataset
This dataset provides character animations for pedagogical agents, generated by replicating instructor behaviors from existing online lecture videos. It is intended for research in educational applications, 3D character animation, and human-agent interaction.  This work is supported by the National Science Foundation (NSF), Research on Emerging Technologies for Teaching and Learning (RETTL) program.
